# Momentum

Momentum is a React Native workout tracker built with Expo Router.

## What This App Does

- Build and manage custom workout routines
- Add exercises with sets, reps, and weight
- Log sessions by marking sets and exercises as complete
- Track workout history and aggregate stats
- Save data locally on device storage

## Run Locally

### Prerequisites

- Node.js `24.11.x` (see `.nvmrc`)
- npm

### Setup

```bash
npm install
```

### Start Development Server

```bash
npm run start
```

From the Expo prompt, open the app on iOS Simulator, Android Emulator, or Expo Go.

### Run Directly Per Platform

```bash
npm run ios
npm run android
npm run web
```

## Useful Commands
